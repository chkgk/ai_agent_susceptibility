Counting Task Instructions

Welcome to the counting task experiment. In this task, you will be presented with a grid containing zeros and ones, and your job is to count how many ones appear in the grid.

What you need to do:

  1. Look carefully at the grid image that will be displayed
  2. Count all the cells that contain the number "1"
  3. Enter your count in the input field provided
  4. Click the "Next" button to submit your answer

Important notes:

  • Take your time to count accurately
  • Each grid is randomly generated, so every participant sees a different pattern
  • Only count the cells containing "1" - ignore the cells with "0"
  • Double-check your count before submitting
Tip: You may find it helpful to count systematically, for example by going row by row or column by column.

Note to researchers: You can customize these instructions by editing the Instructions.html template in the counting_task app.
